A Guide To Charlene

A simple yet useful options panel

While keeping the theme minimal Charlene does boast it's own custom options to make procedures like adding your Google analytics code to the footer file a breeze for anyone. Nothing fancy, all Charlene tries to do is keep the set up of your site as minimal as the look of the theme itself.

Theme options are split into five sections at the moment for organizational purposes: General, Header, Front Page and Footer. For a complete look at the available options please visit http://papertreedesign.com/themes/theme-admin-panel/

Other useful things to note

Widget areas:

  • Sidebar Main

    Main sidebar widget area.

  • Sidebar Secondary

    Loaded after the main sidebar, especially useful if you decide to split and go with two sidebars.

  • After Post

    Widget ready area for after post plugins and functions that are packaged with widgets. Would be useful for recent posts or perhaps a text area that links to your available Rss subscriptions.

SEO

A few special extras have been added to Charlene to improve overall site SEO. However if you choose to use plugins such as All-In-One-SEO Pack you can easily switch off the built in SEO functions so that you are not repeating yourself.

Credit Is Due

Charlene is a theme based on something that I had created in the past for a personal WordPress site on which received some good feedback and even requests for theme release. This revamped version is built on a theme framework I assembled for my own personal/business use. I contemplated whether it was necessary to include all of the features of that framework in a publicly released theme and after some thought decided to go ahead with it as they may be useful to some.

Would not be fair of me to leave out mention a few of the places from which I have learned and received inspiration along the way. The Sandbox Theme, the originals by Brian Gardner, Thematic by Ian Stewart, Hybrid by Justin Tadlock, Thesis by Chris Pearson, WordPress and of course the Kubrick Theme. Many more but you get the idea.
